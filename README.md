This project developed an end-to-end machine learning pipeline to predict podcast engagement and popularity. Starting with data preprocessing, it handled missing values, encoded categorical features, and normalized data using StandardScaler. Tableau visualizations revealed key trends, guiding feature selection. The modeling phase compared Random Forest (RF) and a Simple Neural Network (SNN), with RF emerging as the superior model due to its robustness and interpretability, achieving high accuracy across RMSE, R², and MAE metrics. The final model was deployed via a Streamlit web app hosted on AWS, ensuring scalable and accessible predictions. By leveraging ensemble learning and effective data engineering, this solution provides actionable insights for content creators to optimize podcast performance
